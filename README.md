# Data Structures and Algorithms Projects

This repository contains a collection of projects implemented in **C language** for the course Data Structures and Algorithms.

Each project focuses on applying a specific data structure or algorithm to solve real-world problems.

---

## Projects Overview

### Project 1: Linked Lists & Radix Sort
- Implemented a **doubly linked list** to manage hierarchical data.
- Applied **Radix Sort** to sort districts alphabetically.
- Managed towns within districts and sorted them based on population.
- Included features such as:
  - Adding/removing districts and towns
  - Updating population
  - Calculating statistics (total, min, max population)
- Input/Output handled using files. 

---

### Project 2: Stack (Undo/Redo Text Editor)
- Built a text editor using:
  - **Undo Stack**
  - **Redo Stack**
  - **Queue for insert operations**
- Supported operations:
  - Insert and remove text
  - Undo/Redo changes dynamically
  - Save updated text to file
- Implemented stacks using **linked lists**. 

---

### Project 3: AVL Tree & Hash Table
- Implemented:
  - **AVL Tree** for balanced word storage
  - **Hash Table** for fast lookup
- Features:
  - Word frequency counting
  - Insert/Delete/Search operations
  - Collision handling in hash table
  - Word statistics (most frequent, repeated words)
- Input taken from text files.

---

### Project 4: Graphs & Topological Sorting
- Implemented:
  - **Dijkstra Algorithm** for shortest path between buildings
  - **Topological Sort** for course prerequisite planning
- Features:
  - Find shortest route between campus buildings
  - Sort courses based on prerequisites
- Used graph representations (matrix/list)

---
